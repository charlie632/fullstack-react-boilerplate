# fullstack-react-boilerplate


## Installation
### Clone 
```bash
  git clone https://github.com/charlie632/fullstack-react-boilerplate.git
```

### Change directory
```bash
cd fullstack-react-boilerplate
```

### Install dependencies
```bash
npm i
```

### Bootstrap project
**Mind the `run` word
```bash
npm run install
```


### Open VS Code
```bash
code .
```
